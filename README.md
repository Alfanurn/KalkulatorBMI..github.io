# Spesifikasi Kalkulator BMI 

The BMI value is interpreted into the following categories:

- KURUS: BMI < 18.5
- IDEAL: 18.5 ≤ BMI < 24.9
- TERLALU GEMUK: 24.9 ≤ BMI < 29.9
- OBESITAS: BMI ≥ 30
